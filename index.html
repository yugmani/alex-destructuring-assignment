How Destructuring Assignment in JavaScript Works

  • What is destructuring?

    1. Destructuring is a way to extract values from data and assign those values to one or more variables.
    2. Destructuring works only with arrays and objects.

  • There are two ways to use deconstructing.

    1. First, you can use it to assign a value to a variable when you declare it.
    2. Second, you can declare an empty variable and use destructuring later to assign it a value. When you want to use this, pay attention to the type of variable you use.

  • Destructuring arrays

    1. First, you have to use syntax with square brackets [].
    2. Second, inside these square brackets, you specify the variable name(s) you want to use. When you specify the variables names, make sure you write them in the right order.
